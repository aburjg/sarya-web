Quick start, with hello world

1. First install Sarya

.

pip install sarya

2. Copy and paste this into a python file

.

from sarya import Sarya, UI

sarya = Sarya("xxxx....")

@sarya.publish()
def hello_1234():
  return UI.Text("Hello World!")

sarya.run()

3. Run file

.

python hello.py

.

That's it! Try it on now on Superchat/@hello-1234
