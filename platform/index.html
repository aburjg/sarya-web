Getting started

Install Sarya:

pip install sarya

Create a hello world app

from sarya import Sarya, UI

sarya = Sarya("xxxx....")

def hello():
    return UI.Text("Hello World!")

sarya.run()

then publish it

python main.py

head on to the Superchat and call your app with @hello
